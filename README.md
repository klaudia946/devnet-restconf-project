# DevNet RESTCONF Project

## Author
Klaudia

## Description
This project demonstrates network automation using Python and RESTCONF.

The script connects to a Cisco IOS XE router and automatically creates a network interface configuration.

## Technologies
- Python
- RESTCONF API
- Cisco IOS XE
- Ubuntu
- Git & GitHub

## File
restconf.py – Python script that sends RESTCONF requests to configure an interface.

## Example workflow
Python script → RESTCONF API → Cisco Router → Interface configuration
